# Changelog

## v3.0.0

- 人脉地图从纯 Skill 升级为能力包：新增 `relationship-map-vault` Hermes 插件与捆绑 Skill。
- 新增独立关系资产库，不再把联系人、互动、承诺和待办只依赖于聊天记忆。
- 新增联系人查询、详情、互动沉淀、承诺沉淀、待跟进和会前准备六项受控能力。
- 普通互动、明确承诺和待跟进事项自动沉淀，并在用户回复中显示“已记录到人脉地图”或“已更新到人脉地图”。
- 增加用户范围隔离、追加式互动时间线、审计事件与数据库备份能力。
- 用户数据目录与插件代码目录分离；插件升级不覆盖已有关系资产。

## v2.6.1

基于 v2.6 的正式安装入口基线，作为能力包升级前的可回滚版本。

## v2.6

强化资料解析、深度沉淀、会前准备、会后沉淀、资源路径化、关键人维护建议和关系可调动度。
