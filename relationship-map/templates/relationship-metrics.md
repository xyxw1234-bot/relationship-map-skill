关系指标模板

生成原则

指标按信息充分度选择性生成：有依据才生成，没有依据就隐藏，不要默认给所有联系人打满指标。

核心指标

- 关系温度：冷 / 温 / 热 / 关键期 / 风险期（有最近互动或联系场景时生成）
- 关系强度：弱连接 / 可联系 / 稳定熟人 / 深度信任 / 核心盟友
- 信任程度：低 / 中 / 高 / 极高（有长期互动证据时生成，不要默认写未知）
- 行动优先级：A / B / C / D
- 机会匹配度：低 / 中 / 高
- 风险级别：低 / 中 / 高（有风险信号时生成，无风险信号则隐藏）
- 信息置信度：用户确认 / 用户提供未核验 / 模型推断 / 过期待核验
