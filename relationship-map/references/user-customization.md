用户自定义扩展

人脉地图是可改、可扩展的 Skill，不应锁死。

推荐方式

用户自己的字段、模块、场景和偏好，放在：

$HERMES_HOME/data/relationship-map/extensions/

这样 GitHub 更新官方 Skill 时，不会覆盖用户自定义内容。

可扩展内容

- 联系人字段
- 关系指标
- 行业模块
- 场景剧本
- 历史复杂方案筛选
- 话术风格
- 复盘问题
- 语义排序规则
- 自定义标签体系
- 自定义关系强度和联系频次口径

不可突破的底线

- 不能关键词自动弹卡，必须过语义闸门。
- 敏感写入必须确认。
- 用户数据不能被 Skill 更新覆盖。
- 一级列表不能泄露敏感信息。
- 推断不能写成事实。
